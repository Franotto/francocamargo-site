# Design Specification: Franco Camargo Advocacia (Premium 2026)

## 1. Identidade Visual (Paleta de Cores)

As cores foram convertidas para HEX para garantir consistência em todos os navegadores:

| Elemento       | RGB           | HEX       | Uso                                                        |
| :------------- | :------------ | :-------- | :--------------------------------------------------------- |
| **Primary**    | 116, 44, 45   | `#742C2D` | CTAs, Títulos em destaque, Ícones, Hover links.            |
| **Secondary**  | 55, 52, 53    | `#373435` | Menu (Fundo), Rodapé (Fundo), Texto base.                  |
| **Accent**     | 178, 178, 178 | `#B2B2B2` | Bordas, Divisores, Textos secundários, Ícones desativados. |
| **Background** | 249, 249, 249 | `#F9F9F9` | Fundo principal (Off-white para contraste suave).          |

## 2. Tipografia (Montserrat)

- **H1 (Principais):**
  - Tamanho: 58px
  - Peso: Semi-bold
  - Letter-spacing: 1.1px
  - Uso: Headline do Hero.
- **Subtítulos (H2/H3):**
  - Tamanho: 20px
  - Peso: Normal
  - Line-height: 1.5
  - Letter-spacing: 1.1px
  - Uso: Títulos de seções, nomes de áreas e posts de blog.
- **Corpo do Texto:**
  - Tamanho: 16px
  - Peso: Normal
  - Fonte: Montserrat (Sans-serif)

## 3. Elementos Gráficos

- **Ícones:** LucideIcons ou FeatherIcons.
- **Estilo:** Stroke width de 1.5px (fino) para manter a leveza e sofisticação.
- **Fotografia:** Imagens de arquitetura jurídica, espaços corporativos modernos e retratos profissionais com filtro leve (preto e branco ou desaturado).
